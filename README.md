# Glowstone Website

Premium engineered quartz surfaces website for Glowstone Industries. 93% pure natural quartz, mineral-infused, anti-microbial certified.

## Contents

- **HTML pages**: Home, About, Collections, Contact, FAQ, Installation, Projects, Sustainability, Technical Specifications, Care & Maintenance
- **Collections**: Domestic (Calacatta, Carrara, Onyx, Pastel, Budget, Plain) and International
- **Assets**: CSS, JavaScript, images, certificates, PDFs

## Note on Video

The `video/` folder (containing the 1.8GB factory tour video) is excluded from this repository due to GitHub's 100MB file limit. Host the video separately (e.g., YouTube, Vimeo, or your CDN) and update the embed URL in your HTML.

## Local Development

Open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit http://localhost:8000
